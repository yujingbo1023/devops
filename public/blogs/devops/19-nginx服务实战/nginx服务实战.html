    19-nginx服务实战

    # 1，nginx虚拟主机

    # a，基于多域名的方式

    1.基于多域名的方式
1)www.xbw.com ----->小霸王游戏
第一步: 修改nginx的配置文件
[root@web01:conf.d]#cat xbw.conf
server {
   listen 80;
   server_name www.xbw.com;

   location / {
      root /code/xbw;			# 存放代码的目录
      index index.html;
   }

}

第二步: 创建代码目录
[root@web01:conf.d]#mkdir -p /code/xbw
[root@web01:conf.d]#cd /code/xbw

第三步: 上传游戏代码,解压
[root@web01:xbw]#ll
total 7720
-rw-r--r-- 1 root root 7902976 Aug  6 10:04 小霸王_FC怀旧游戏机-HTML源码.zip

[root@web01:xbw]#unzip 小霸王_FC怀旧游戏机-HTML源码.zip

第四步: 检查nginx配置
[root@web01:xbw]#nginx -t
nginx: the configuration file /etc/nginx/nginx.conf syntax is ok
nginx: configuration file /etc/nginx/nginx.conf test is successful



第五步: 重启nginx
[root@web01:xbw]#systemctl restart nginx

第六步：配置dns解析
10.0.0.7  www.xbw.com

浏览器（www.xbw.com）测试xbw游戏。

=======================================================================

2)配置kunkun的业务
第一步: 修改nginx配置
[root@web01:conf.d]#cat kunkun.conf

[root@web01:conf.d]#mkdir /code/kunkun

[root@web01:conf.d]#cd /code/kunkun/

第六步: 浏览器（www.kunkun.com）测试kunkun游戏。

C:\Windows\System32\drivers\etc

10.0.0.7 www.xbw.com www.kunkun.com

修改完成后,将hosts文件在拖拽回源路径
测试是否解析成功:
打开cmd运行
